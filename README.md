Webhook example: 
{
  "name": "Webhook",
  "enable": true,
  "urls": [you can use urls from https://webhook.site]
}

For trigger example:
{
  "data": {
    "userId": 123,
    "action": "create"
  }
}

You can access the API documentation via Swagger UI at the following URL: http://localhost:3000/api/webhooks.
